This application is for tracking personal expenses during the month. 

You can add and remove expenses. In this case, you need to enter the title and amount, as well as select a category and date, if necessary. 

A diagram is provided for cost analysis. It also allows you to view all expenses of any category.
On the last day of the month, it is advisable to clear all expenses and take a chart screenshot to compare it with other months results later.

No more than 100 expenses - this is just a light touch of the game, may be allowing you to save money.
